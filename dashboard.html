Prediction Task

Given the last 60 days of multi-asset market data, predict whether BTC will close HIGHER (↑) or LOWER (↓) the next day.

label_dir_1d = (close_btc.shift(-1) > close_btc).astype(int)

1 → BTC price goes UP tomorrow
0 → BTC price goes DOWN tomorrow

Key Feature Insights
Cross-Asset Returns
Gold and DXY returns have the highest correlation with BTC direction, confirming the macro influence identified in the 2025 literature (Köse et al.).
MVRV Ratio
Fundamental valuation metric. MVRV > 3.7 historically signals tops; MVRV < 1 signals bottoms. Strong non-linear signal.
Funding Rate
Positive rate = longs pay shorts (bullish bias). Extreme values predict reversals. Averaged across 3 daily settlement times.
Day-of-Week Encoding
Cyclic sin/cos encoding preserves the circular nature of weekdays (Monday ≈ Sunday). Bitcoin shows measurable weekend effects.

Model Architectures

7 models trained: 6 PyTorch deep learning + 1 gradient boosting

LSTM
Recurrent Neural Network · PyTorch
Long Short-Term Memory with 3 gates (forget, input, output). Retains information selectively over long sequences. The most studied architecture for crypto prediction in the literature.
hidden: 64 layers: 1 dropout: 0.3 seq_len: 60
BiLSTM
Bidirectional LSTM · PyTorch
Processes sequence in both directions (past→future and future→past). Better representation of each timestep. Doubles parameters versus standard LSTM. Useful for capturing symmetric patterns.
hidden: 64×2 layers: 1 bidirectional: True seq_len: 60
GRU
Gated Recurrent Unit · PyTorch
Simplified LSTM that merges forget and input gates into a single update gate. ~25% fewer parameters than LSTM. Trains faster with comparable performance on short-to-medium sequences.
hidden: 64 layers: 1 dropout: 0.3 lighter than LSTM
CNN-LSTM
Hybrid Convolutional + Recurrent · PyTorch
Conv1D layers extract local temporal patterns (candlestick shapes, volume spikes), then LSTM contextualizes globally. Particularly effective when data contains strong technical patterns.
filters: 32 kernel: 3 lstm_hidden: 64 2-stage
Transformer
Attention-Based Encoder · PyTorch
Multi-head self-attention over the entire sequence. Each timestep can directly "attend" to any other. Sinusoidal positional encoding. Competitive but data-hungry.
d_model: 64 heads: 4 enc_layers: 2 ff_dim: 128
TFT
Temporal Fusion Transformer · PyTorch · SOTA
State-of-the-art multi-variate time series model. Combines: Variable Selection Network, LSTM encoder, interpretable multi-head attention, and Gated Residual Networks. Automatically weights feature importance.
hidden: 64 lstm_layers: 1 attn_heads: 4 SOTA per literature
XGBoost
Gradient Boosting · sklearn wrapper
Flattens 3D sequences (60 days × features) into 2D vectors. Ensemble of decision trees with L1/L2 regularization. Strong baseline — if DL doesn't beat it, complexity isn't justified.
estimators: 500 max_depth: 6 lr: 0.05 objective: binary:logistic

Key Findings from Literature
TFT dominates (Köse et al., 2025)
Gold price identified as the most important macro factor for BTC prediction. TFT outperforms LSTM/GRU/SVR/XGBoost on 5 cryptocurrencies in RMSE, MAE, and R².
CNN-LSTM + Boruta = 82% (ScienceDirect, 2025)
Feature selection with Boruta algorithm before CNN-LSTM consistently outperforms other combinations. Best performer on intraday data with strong technical patterns.
LSTM benchmark (Critien et al., 2024)
In systematic comparison across 5 cryptos, LSTM achieves best RMSE (0.0222), 2.7% better than second model. RNNs occupy the top 3 slots ahead of CNN and Transformers.
Efficient market caution
Theoretical market efficiency suggests prediction should be impossible. In practice, micro-inefficiencies exist but degrade quickly once models are deployed (strategy decay).
Priority Recommendations (Literature Consensus)
1
TFT (Temporal Fusion Transformer) — Best performance/interpretability ratio. Learns which features and timesteps matter. Use when multiple heterogeneous feature types are available.
2
LSTM / BiLSTM — Most robust and best-documented choice. Excellent starting point. Strong literature backing for BTC specifically.
3
CNN-LSTM — Best when data contains strong technical patterns (intraday with candlestick patterns, volume spikes).
4
XGBoost baseline — Always train as baseline. If deep learning doesn't beat it significantly, added complexity is not justified.
